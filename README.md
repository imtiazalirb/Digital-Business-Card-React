## Run Locally

Clone the project

```bash
  git clone /https://github.com/imtiazalirb/Digital-Business-card-React.git
```

Go to the project directory

```bash
  cd Digital-Business-card-React
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
